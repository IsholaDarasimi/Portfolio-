Courses & Workshops

Each course is project-based. I run short workshops or multi-week classes depending on school needs.

HTML Essentials — Build a Live Website

Description: Start with HTML structure, elements, forms, accessibility basics, then add CSS layout. Students build a multi-page mini-site they can host or zip and submit.

  • Duration: 4–6 lessons (1–1.5 hrs each)
  • Outcome: 4-page mini-site with contact form and gallery
  • Skills: semantic HTML, basic accessibility, responsive layout

Digital Awareness & Online Safety

Description: Age-adjusted sessions on privacy, safe social media, phishing awareness, and responsible internet use. Includes parent-facing checklist for at-home safety.

  • Duration: 2–3 lessons
  • Outcome: Student checklist & teacher brief
  • Skills: account safety, privacy settings, cyber hygiene

Video Editing Basics

Description: Shot composition, basic editing workflow, transitions, titles, export for web. Students produce a 30–60s video by course end.

  • Duration: 3–4 lessons
  • Outcome: Polished short video + export presets
  • Skills: storyboarding, trimming, layering audio
